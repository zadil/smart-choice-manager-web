Politique de Confidentialité

Dernière mise à jour : 30 juillet 2024

1. Introduction

Smart Choice Manager ("nous", "notre", "nos") respecte votre vie privée et s'engage à protéger vos données personnelles. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre add-on Google Workspace.

2. Données Collectées

2.1 Données de Formulaire

Notre add-on accède uniquement aux données suivantes dans votre environnement Google Workspace :

  • Structure et contenu de vos formulaires Google Forms
  • Réponses soumises à vos formulaires
  • Paramètres de configuration que vous définissez dans l'add-on
  • Données de suivi stockées dans les feuilles Google Sheets associées

2.2 Données Techniques

Nous ne collectons aucune donnée technique ou analytique. L'add-on fonctionne entièrement dans votre environnement Google Workspace.

3. Utilisation des Données

Les données sont utilisées exclusivement pour :

  • Gérer les limites de réponses dans vos formulaires
  • Suivre le nombre de réponses pour chaque option
  • Mettre à jour automatiquement la disponibilité des choix
  • Fournir des fonctionnalités de configuration et de gestion

4. Stockage et Sécurité

4.1 Localisation des Données

Toutes vos données restent dans votre environnement Google Workspace. Aucune donnée n'est transmise vers des serveurs externes ou des services tiers.

4.2 Sécurité

La sécurité de vos données est assurée par :

  • L'infrastructure sécurisée de Google Workspace
  • Les protocoles de sécurité standard de Google Apps Script
  • L'accès limité aux seules données nécessaires au fonctionnement
  • Aucune transmission de données vers l'extérieur

5. Partage des Données

Nous ne partageons, ne vendons, ni ne transmettons vos données à des tiers. Vos données restent sous votre contrôle total dans votre environnement Google Workspace.

6. Autorisations Requises

L'add-on demande les autorisations suivantes :

  • Google Forms : Pour lire et modifier la structure de vos formulaires
  • Google Sheets : Pour créer et gérer les feuilles de suivi des réponses
  • Google Drive : Pour accéder aux fichiers de formulaires et feuilles associés
  • Interface utilisateur : Pour afficher les menus et dialogues de configuration

7. Vos Droits

Vous conservez un contrôle total sur vos données :

  • Accès : Toutes vos données sont accessibles dans vos formulaires et feuilles Google
  • Modification : Vous pouvez modifier ou supprimer vos données à tout moment
  • Suppression : Vous pouvez désinstaller l'add-on et supprimer toutes les données associées
  • Portabilité : Vos données restent dans des formats standard Google

8. Conformité

Notre add-on respecte :

  • Les conditions d'utilisation de Google Workspace
  • Les politiques de sécurité de Google Apps Script
  • Le Règlement Général sur la Protection des Données (RGPD)
  • Les réglementations locales sur la protection des données

9. Modifications de cette Politique

Nous pouvons mettre à jour cette politique de confidentialité occasionnellement. Les modifications seront publiées sur cette page avec une date de mise à jour révisée. Nous vous encourageons à consulter régulièrement cette politique.

10. Contact

Si vous avez des questions concernant cette politique de confidentialité ou le traitement de vos données, vous pouvez nous contacter :

Email : support@smartchoicemanager.com

Documentation : GitHub Repository

11. Désinstallation

Pour désinstaller l'add-on et supprimer toutes les données associées :

  1. Ouvrez Google Forms
  2. Accédez aux Add-ons dans le menu
  3. Sélectionnez "Gérer les add-ons"
  4. Trouvez Smart Choice Manager et cliquez sur "Supprimer"
  5. Supprimez manuellement les feuilles de suivi créées si vous le souhaitez
